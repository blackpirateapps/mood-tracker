API Documentation

Base URL

https://yourdomain.com/api/external

Authentication

Include your API token in the Authorization header:

Authorization: Bearer YOUR_API_TOKEN

GET Fetch All Mood Data

Retrieves all your journal entries and activities

curl -H "Authorization: Bearer YOUR_TOKEN"
https://yourdomain.com/api/external
Response Example
{
  "entries": [
    {
      "id": 1,
      "date": "2025-11-10",
      "mood": 4,
      "activities": ["work", "exercise"]
    }
  ],
  "activities": [...]
}

POST Create Mood Entry

Add a new mood journal entry (requires write permission)

curl -X POST https://yourdomain.com/api/external
-H "Authorization: Bearer YOUR_TOKEN"
-H "Content-Type: application/json"
-d '{
  "action": "save_entry",
  "date": "2025-11-10",
  "dateKey": "2025-11-10",
  "mood": 4,
  "activities": ["activity_123"]
}'

POST Delete Mood Entry

Delete an existing entry (requires write permission)

curl -X POST https://yourdomain.com/api/external
-H "Authorization: Bearer YOUR_TOKEN"
-H "Content-Type: application/json"
-d '{
  "action": "delete_entry",
  "entryId": 123
}'

Rate Limits

API calls are limited to 1000 requests per day per token.
